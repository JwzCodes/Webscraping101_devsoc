# Webscraping101 
A short introduction to webscraping with python using beautiful soup 

# Workshop Overview
Over this workshop we will teach you how to web scrape from he ground up with a focus on the basics of webscraping. From the fundumentals of how websites work and basic HTTP requests to more advanced HTML parsing and information gathering techniques. Participants will build a webscraper to scrape a simulated book store then move onto real world challenges such as scraping a Wikipedia page. 

# Learning objectives
By the end of this workshop, you will be able to:
- Parse HTML content using BeautifulSoup
- Extract stuctured data using regular expressions
- Handle endgecases
- Build a complete web scraping application with user interaction

# Prerequisites 
- Basic Python Knowlege
- Familiarity with command line usage
- A text editor or IDE (VS Code recomended)

# Setup Insstructions
1. Clone or download this workshop respository
2. Create a virtual environment
```
python3 -m venv .venv
. .venv/bin/activate  # On Windows: .venv\Scripts\activate
```
4. Install dependencies
```
python3 -m pip install -r requirements.txt
```
5. Activate the virtual environmenrt
```
source .venv/bin/activate
```
